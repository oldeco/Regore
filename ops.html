Override risk score
Manually set a risk score for this agent. The override will be stored in the Regore database and used until cleared. The automated scoring engine will resume once the override is removed.
0–39 = low · 40–64 = moderate · 65–100 = high risk
Flag agent
Flagging marks this agent in the Regore database for heightened monitoring. It does not stop the agent — only the operator can do that. A flag triggers increased anomaly sensitivity and notifies the DAO review queue.
